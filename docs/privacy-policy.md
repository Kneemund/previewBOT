# previewBOT Privacy Policy

This Privacy Policy describes how data is handled when you use previewBOT and should be read together with the [Terms of Service](terms-of-service.md).

## Data Processed
previewBOT does not permanently store user data or any data derived from user data.

- previewBOT may temporarily process data provided via Discord application commands and message content solely to perform its functionality.
- previewBOT may temporarily cache data provided via Discord application commands to improve performance.

## Data Storage and Control
All user content is stored by Discord, not by previewBOT. previewBOT does not maintain independent databases or long-term storage.

## Data Deletion
User data cannot be deleted upon request, because previewBOT does not store user data. Any deletion must be performed through Discord.

## Third-Party Platform
Use of previewBOT is subject to Discord’s own terms and privacy practices.
